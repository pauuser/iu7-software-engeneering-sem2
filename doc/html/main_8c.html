Detailed Description

Основная программа обработки чисел из файла. На вход подается число 1 и имя файла.

Author
Павел Иванов, ИУ7-25Б

Постановка задачи: В текстовом файле содержатся целые числа. Найти сумму цифр каждого числа и если она четная, то перевести его в двоичную систему счисления, а если нечетная, то преобразовать число в обратном порядке следования цифр. Результат вывести на экран.

Программа подготовлена для домашнего задания по ОПИ (1 курс 2 семестр) 2021 год
